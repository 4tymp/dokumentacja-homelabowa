# Witam w moim dzienniku homelabów!
bede tutaj wlasnymi slowami opisywal moje starania z ogolnopojetego homelabbingu. bede pisal wlasnym stylem i slowami, glownie po to zebym sam za x lat mogl odtworzyc to co sie tutaj działo, oraz co najwazniejsze : **jednoczesnie się uczyć i dobrze bawić!**

niektóre mniejsze tematy bede po prostu pisal w formie poradnika, inne jak pelnoprawna dokumentacje labową, zalezy od tematu.
zawsze bede jednak je zaczynal formą:
```markdown
Tagi: [[tag1]] [[tag2]] [[tag3]]
Data: xx-xx-xxxx
```
tagi są dla mnie, dokumentację prowadzę w **Obsidian**, więc chcę mieć wszystko ładnie połączone. data raczej sama z siebie jest zrozumiała.
